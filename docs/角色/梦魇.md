# 梦魇

> 永恒梦魇笼罩战场，以暗影斩击和恐惧控制为核心，剥夺敌人的技能选择并遮蔽战场信息。
>
> 通用规则（等级结算、生命结构、格挡等）见 [基础规则手册](../基础规则手册.md)。资源与 Buff 速查见 [附录](../附录/)。

## 属性

- **形态名称**：永恒梦魇
- **生命结构**：2 HP（健康 / 濒死 / 死亡）

## 被动技能：暗影之刃

- 无法使用拳和斩（不在技能树中）。
- 开局即拥有一次暗影之刃使用次数。
- 每次使用暗影之刃后进入 4 层冷却。此后每完成一次梦魇专属招式，冷却层数 −1。
- 冷却归零后可再次使用暗影之刃。

## 变身收益

保留全部基础招式（拳和斩除外），额外解锁：

治疗、**暗影之刃**、**魇之梦径**、**黑暗庇护**、**无言恐惧**、**鬼影重重**。鬼影冲刺由鬼影重重授予的 Buff 临时提供。

## 专属技能

### 暗影之刃

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 无 |
| 速度 | 2 |
| 等级 | 1.5 |
| 伤害类型 | 斩击 |
| 目标 | 自身格及顺、逆时针相邻格内的所有其他存活玩家（自动） |
| 解锁条件 | 不在冷却中 |

**结算说明**：对自身所在格及顺时针、逆时针各一格（共三个地块）上的所有其他存活玩家造成 1.5 级斩击。使用后进入 4 层冷却；魇之梦径、黑暗庇护、无言恐惧、鬼影重重和鬼影冲刺各使冷却 −1。

### 魇之梦径

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 1 气 |
| 速度 | 1 |
| 等级 | 1 |
| 伤害类型 | 魔法 |
| 目标 | 1 名其他存活玩家（预选路径终点）；顺时针或逆时针方向；路径内 1 个移动地块 |

**结算说明**：

1. **路径选择与伤害**：选定玩家终点后，再选择顺时针或逆时针方向。路径包含自身格和终点格，对路径上的所有其他存活玩家造成 1 级魔法伤害。
2. **铺设梦径**：路径上的每一格生成持续 3 回合的「魇之梦径」地块。重复铺设同一格时刷新持续时间，不叠加层数。
3. **共享增幅**：任何梦魇位于任意玩家铺设的梦径地块上时，攻击技能等级与伤害等级均提高 0.5。
4. **回合末移动**：梦魇可移动至本次路径上的任意一格，包括自己脚下和已有角色的地块。

### 黑暗庇护

| 属性 | 值 |
|------|-----|
| 分类 | 防御 |
| 消耗 | 1 蓄力 |
| 速度 | 4 |
| 等级 | ∞（吸收判定） |
| 目标 | 自身（自动） |

**结算说明**：

- 本回合只吸收一次攻击——自动选择本回合指向自身的最高等级攻击进行完全免疫。
- 吸收成功后获得「黑暗庇护」Buff（角色级，持续 4 回合含当前）：速度 +1，攻击等级 +0.5。
- 对于其他未被吸收的攻击，自身招式等级计为 0 进行防御结算。
- 若本回合无人攻击梦魇，无任何收益。

### 无言恐惧

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 1 气 + 1 蓄力 |
| 速度 | 1 |
| 等级 | 2 |
| 伤害类型 | 魔法 |
| 目标 | 1 名其他存活玩家（预选） |

**结算说明**：

1. 无言恐惧本身不造成伤害；其 2 级数值只用于控制判定。
2. 若目标本回合对梦魇有效的招式等级 ≤ 2，则附加恐惧效果：
   - **本回合**：目标尚未结算的非气行动被取消（已结算的不受影响）；目标的格挡被移除。
   - **恐惧 Debuff**（玩家级，持续 2 回合）：目标只能使用「气」（提交其他技能会在验证阶段被拒绝）；目标受到攻击时攻击方等级 +1。

### 鬼影重重

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 1 气 + 3 蓄力 |
| 速度 | 2 |
| 等级 | 3 |
| 伤害类型 | 魔法 |
| 目标 | 1 名其他存活玩家（后发；本回合可不选择） |

**结算说明**：

1. **全场黑暗**：除施术者自己以外的所有玩家获得「黑暗」Buff（玩家级，持续 2 回合含本回合）。处于黑暗中的玩家无法看到除自身地块外其他玩家的状态变化与招式。
2. **冲刺锁定**：梦魇获得「鬼影锁定」Buff（角色级，持续 2 回合），可发动鬼影冲刺。
3. **立即冲刺**（可选）：后发阶段可选择一名目标并立即发动冲刺攻击（3 级）。击杀目标则位移至目标地块；未击杀则位移至目标顺时针相邻格（被占则留在原地）。
4. **保留冲刺**：可在后发阶段跳过目标选择，不发动攻击和位移，但仍造成全场黑暗和获得鬼影锁定。可在下一回合使用鬼影冲刺消耗保留的锁定——总窗口内只能冲刺一次。

### 鬼影冲刺

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 无 |
| 速度 | 2 |
| 等级 | 3 |
| 伤害类型 | 魔法 |
| 目标 | 1 名其他存活玩家（预选） |
| 获得方式 | 鬼影重重授予的「鬼影锁定」Buff 临时提供此技能 |

**结算说明**：消耗鬼影重重保留的锁定，对目标发动 3 级冲刺攻击。位移规则与鬼影重重的立即冲刺相同。

## 设计定位

梦魇通过专属招式推进暗影之刃冷却；通用招式不推进冷却。
