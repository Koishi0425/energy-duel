# 皮卡丘

> 以雷电场控与传导攻击为核心，依靠棋盘位置判断命中，并具备基础位移能力。
>
> 通用规则（等级结算、生命结构、格挡等）见 [基础规则手册](../基础规则手册.md)。资源与 Buff 速查见 [附录](../附录/)。

## 属性

- **形态名称**：雷电形态
- **生命结构**：2 HP（健康 / 濒死 / 死亡）
- **被动技能**：无

## 变身收益

保留全部基础招式，额外解锁：

拳、斩、治疗、**十伯伏特**、**十万伏特**、**迅雷**。

## 核心机制：双向传导

十伯伏特和十万伏特不使用传统的目标选择，而是沿棋盘两个方向逐格搜索命中：

1. 从自身地块出发，分别沿顺时针和逆时针方向逐格检查。
2. 每个方向上命中第一个存活的其他玩家。
3. 两个方向分别独立结算——可能命中两个不同的玩家，也可能同一玩家在两个方向都是第一个被找到（此时只命中一次）。

## 专属技能

### 十伯伏特

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 1 蓄力 |
| 速度 | 1 |
| 等级 | 1 |
| 伤害类型 | 魔法 |
| 目标 | 顺、逆时针方向各第 1 名其他存活玩家（自动） |

**结算说明**：沿顺、逆时针逐格传导，各命中第一个未躲开的玩家，造成 1 级魔法伤害。若持有「迅雷余势」Buff 则不消耗蓄力。

### 十万伏特

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 2 气 + 2 蓄力 |
| 速度 | 1 |
| 等级 | 3 |
| 伤害类型 | 魔法 |
| 目标 | 顺、逆时针方向各第 1 名其他存活玩家（自动） |

**结算说明**：与十伯伏特相同，使用双向传导机制选择目标。

### 迅雷

| 属性 | 值 |
|------|-----|
| 分类 | 特殊 |
| 消耗 | 1 蓄力 |
| 速度 | 2 |
| 等级 | 0 |
| 目标 | 两人局为 1 个相邻空地块；三人及以上为任意空地块（预选） |

**结算说明**：两人局移动到顺时针或逆时针相邻的一个空置地块；三人及以上对局可移动到任意空置地块。若目标格被占用则留在原地。获得「迅雷余势」Buff（角色级），下一次十伯伏特不消耗蓄力。迅雷按速度 2 结算；与攻击同速时位移优先，可离开普通攻击预先锁定的地块，但不能躲开速度更高或明确锁定玩家的攻击。

## 设计定位

皮卡丘的传导攻击不依赖目标选择，而是由站位决定。这使得他可以通过迅雷调整位置来改变传导路径。在多人局中，巧妙站位可以让传导跳过某些玩家去命中后排目标。伤害类型「魔法」用于未来可能的属性克制系统，当前与通用伤害无区别。
