# 奎隆，摩诃萨埵权化

> 尼卢火增幅攻击焚尽战场，托生莲座巡回收割，惩五戒一击定生死。游戏开始累积资源觉醒无忧觉，死后复活化菩萨辩。
>
> 通用规则（等级结算、生命结构、格挡等）见 [基础规则手册](../基础规则手册.md)。资源与 Buff 速查见 [附录](../附录/)。

## 属性

- **形态名称**：摩诃萨埵权化
- **生命结构**：2 HP（健康 / 濒死 / 死亡）
- **被动技能**：无忧觉（玩家级）、菩萨辩（复活后获得）

## 被动技能

### 无忧觉

- 从本局游戏开始追踪该玩家累计获得的气与蓄力总量（消耗不扣减累计值；未变身为奎隆时也持续追踪）。
- 累计总量 ≥ 5 时，永久满足「承三身」的使用条件。
- 奎隆在回合中受到致死伤害时，若无忧觉尚未消耗，则先完成该回合剩余的全部伤害结算，最后统一触发复活：恢复至健康状态（2 HP），保留死亡时所在地块，移除所有负面 Buff。复活前视为死亡，不能继续结算自己的未执行行动；复活后无忧觉永久失效，获得菩萨辩被动。
- 此复活每局限 1 次。若已消耗，死亡正常结算。

### 菩萨辩

获得方式：无忧觉复活后自动获得。

- 奎隆所有攻击的技能等级 +0.5，伤害等级不增加。
- 拳变为 2 段连击。第一段造成伤害（含 +0.5 技能等级），尼卢火以攻击增幅模式叠加到该段伤害等级上。第二段不造成伤害，改为独立触发尼卢火：每个覆盖目标的尼卢火独立造成 1 次 0.5 伤害，伤害属性与类型跟随拳；多次触发遵循通用规则取最大有效伤害等级。
- 斩变为 2 段连击。第一段造成伤害（含 +0.5 技能等级），尼卢火以攻击增幅模式叠加到该段伤害等级上。第二段不造成伤害，改为独立触发尼卢火：每个覆盖目标的尼卢火独立造成 1 次 0.5 伤害，伤害属性与类型跟随斩；多次触发遵循通用规则取最大有效伤害等级。

## 变身收益

保留全部基础招式，额外解锁：

治疗、**呼吸法**、**惩五戒**、**清火执**。无忧觉累计 ≥ 5 时可使用**承三身**。

## 核心机制：尼卢火

尼卢火是奎隆通过呼吸法布置的永久地块效果，具有以下通用规则：

- **布置**：尼卢火是类似国度的地块效果，可放置在棋盘上任意没有存活玩家或召唤物的地块；其他地块效果不妨碍布置。每名奎隆在同一地块至多保留 1 个自己的尼卢火；尼卢火永久存在，直至被移除或奎隆死亡。
- **覆盖范围**：每个尼卢火覆盖自身所在地块及顺时针相邻、逆时针相邻各 1 格（共 3 个地块）。
- **攻击增幅**：奎隆每次使用攻击分类技能结算时，对于该次攻击的每个目标，分别计算覆盖该目标所在地块的尼卢火数量。每个覆盖该地块的尼卢火使该次攻击对该目标的伤害等级 +0.5。此加成在技能对抗之后、防御减免之前应用。尼卢火伤害属性和伤害类型跟随触发攻击的伤害属性和伤害类型。
- **伤害减免**：每个尼卢火使奎隆受到的所有伤害等级 −0.5。多段攻击的每段分别计算减免。
- **抵抗**：场上存在至少 1 个尼卢火时，奎隆获得「抵抗」效果——不受其他玩家施加的负面 Buff 影响。抵抗为二值状态（有或无），不随尼卢火数量叠加。
- **移除**：任何存活玩家可以使用治疗并选择 1 个尼卢火所在地块为目标；此次治疗不恢复健康状态，改为移除该地块上的 1 个尼卢火；此次治疗不触发脆弱状态。此操作视为治疗行动（分类为特殊，速度 0，目标为地块）。

## 专属技能

### 呼吸法

| 属性 | 值 |
|------|-----|
| 分类 | 特殊 |
| 消耗 | n 点任意资源（气与蓄力混合支付） |
| 速度 | 1 |
| 等级 | 0 |
| 目标 | n 个没有单位的地块（预选，不得重复选择同一地块） |

**结算说明**：

1. 提交时选择 n（1 ≤ n ≤ 当前持有气与蓄力之和），并为每个支付点选择 1 个合法地块。
2. 在每个所选地块上布置 1 个尼卢火。该地块不能有存活玩家或召唤物；国度、尼卢火等地块效果不妨碍布置。
3. 同一名奎隆不能在同一地块重复布置自己的尼卢火。

### 惩五戒

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 3 气 |
| 速度 | 1（若对方攻击技能速度更高，则提升至与对方一致） |
| 等级 | 3（若对方攻击技能等级更高，则提升至与对方一致；此外每个尼卢火使技能等级 +0.5） |
| 伤害类型 | 斩击 |
| 目标 | 1 名其他存活玩家（预选） |
| 通用规则 | 多段攻击通则（基础规则手册第 4.2 节），技能对抗仅判定一次 |

**结算说明**：

1. 提交时确定目标。技能对抗使用匹配并加成后的最终技能等级，仅判定一次。
2. 若最终技能等级大于目标的对抗技能等级，则 5 段攻击全部命中。每段基础伤害等级依次为 1、1、2、2、3。各段分别加上覆盖目标的尼卢火数量 × 0.5 得到该段最终伤害等级。
3. 若最终技能等级小于或等于目标的对抗技能等级，则全部 5 段不命中。
4. 伤害结算遵循多段攻击通则——各段独立判定伤害，健康结算仅使用最大有效伤害等级。

### 清火执

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 1 气 |
| 速度 | 1 |
| 等级 | 1.5 |
| 伤害类型 | 真实 |
| 目标 | 所有尼卢火（自动） |

**结算说明**：

1. 场上所有尼卢火同时触发：每个尼卢火对其自身所在地块及顺时针、逆时针各 2 格（共 5 个地块）上的其他存活玩家造成 1.5 真实伤害。
2. **特殊规则**：多个尼卢火的伤害范围重叠时，同一玩家受到的总伤害等级 = 覆盖该玩家的尼卢火数量 × 1.5。此规则为清火执专属，不改变通用最大伤害规则。
3. 清火执不触发尼卢火的攻击增幅。

### 承三身

| 属性 | 值 |
|------|-----|
| 分类 | 特殊 |
| 消耗 | 2 蓄力 |
| 速度 | 1 |
| 等级 | 0 |
| 目标 | 1 个方向（顺时针或逆时针，预选） |
| 解锁条件 | 无忧觉累计 ≥ 5 |

**结算说明**：

1. 在奎隆当前所在地块召唤 1 个托生莲座实体。莲座拥有 10 生命值，可与该地块上的其他单位共存。
2. 莲座每回合以速度 4 自动向所选方向移动 2 格。移动过程中经过的每个地块依次检查是否有存活玩家；经过的第 1 格和第 2 格分别独立触发吸收。
3. 莲座移动至有存活玩家的地块时，立即分别吸取该地块每名存活玩家当前持有的一半资源（气与蓄力分别计算，向上取整）。成功吸取后，莲座速度永久降为 1、每回合移动 1 格。
4. 莲座回到奎隆召唤时的起始地块时立即消失，奎隆获得莲座携带的全部资源。
5. 任何存活玩家可以使用攻击分类技能选择莲座为目标。莲座不参与技能对抗，直接承受伤害。莲座被击杀时，其携带的全部资源返还给被吸取的原玩家。
6. 奎隆死亡时，莲座立即消失，携带的资源返还。

## 设计定位

奎隆是召唤物体系运营角色。核心循环分为两个阶段：

**第一阶段（复活前）**：使用呼吸法灵活支付任意资源批量布置尼卢火构建阵地 → 尼卢火提供伤害减免和抵抗自保 → 惩五戒的高额多段攻击叠加火伤增幅打出致命伤害 → 清火执在火群密集时提供高额叠加范围真实伤害。此阶段奎隆需要在布置火群和消耗资源之间平衡——惩五戒消耗 3 气极高，呼吸法批量布置也消耗大量资源。

**第二阶段（复活后）**：无忧觉累计满 5 资源解锁承三身 → 托生莲座快速巡游（每回合 2 格）吸取对手资源 → 死亡后复活获得菩萨辩 → 菩萨辩使拳斩变为多段，进一步提升尼卢火触发频率。

奎隆的主要弱点：
- 尼卢火可被对手用治疗移除，对手可以主动清理火群阵地；
- 惩五戒若技能等级不足以压制对方则完全失效，风险极高；
- 布置尼卢火需要消耗资源，前期节奏较慢；
- 复活仅一次，且复活后无忧觉永久失效；
- 托生莲座可被攻击摧毁，资源返还给原主。
