# 内卫

> 邪魔役使的国度守卫，以三枚装置替代常规生命，受伤时布置永久的国度场地，濒死时触发不破无敌自保。
>
> 通用规则（等级结算、生命结构、格挡等）见 [基础规则手册](../基础规则手册.md)。资源与 Buff 速查见 [附录](../附录/)。

## 属性

- **形态名称**：邪魔护卫
- **生命结构**：装置制（详见核心机制）
- **被动技能**：使役邪魔

## 变身收益

内卫不能使用拳、斩和治疗。可使用其余通用基础行动，并额外解锁：

**溃散**、**坍缩恐惧**。

## 核心机制：装置与国度

### 装置制生命

内卫不使用标准健康/濒死/死亡结构，改为装置计数：

- 初始拥有 3 个装置。装置计数为角色级（切走保留、切回恢复）。
- 持有 2 或 3 个装置时视为健康状态。
- 持有 1 个装置时视为濒死状态。
- 持有 0 个装置时进入死亡状态。

**伤害结算**：

结算时先确认攻击是否仍产生至少 0.5 级有效伤害，再根据伤害来源的原伤害等级判定装置损失（同一回合仍只采用最严重的一次结果）：

| 伤害条件 | 装置损失 |
|---|---|
| 有效伤害等级 < 0.5 | 0 |
| 有效伤害等级 ≥ 0.5，且原伤害等级 < 3 | 1 |
| 有效伤害等级 ≥ 0.5，且原伤害等级 ≥ 3 | 2 |

因此，即使技能对抗将一个原伤害等级为 3 的攻击压低，只要该攻击仍产生至少 0.5 级有效伤害，内卫便请求损失 2 个装置，而不是按对抗后的差值只损失 1 个。

**装置锁定**：

若一次伤害结算导致内卫的装置数将降至 ≤ 1，则改为将装置数锁定为 1，且内卫在下回合获得「不破」效果（防御等级视为 +∞，持续 1 回合）。此效果每次从 ≥ 2 装置受击降至 ≤ 1 时均触发。

**左移效果与脆弱状态**：

- 直接声明“左移健康状态”的效果（如剁）使内卫损失 1 个装置。
- 直接左移和脆弱状态造成的装置损失都必须经过装置锁定判定：若内卫原本持有至少 2 个装置，且本次结算会使装置数降至 1 个或更少，则改为锁定为 1 个装置，并在下回合获得「不破」。
- 内卫仅剩 1 个装置时不再满足装置锁定条件；此时再次损失装置会正常死亡。

### 国度

每当内卫损失装置时，在伤害来源所在地块及该地块顺时针、逆时针各相邻一格（共 3 个地块）上各生成 1 个永久的「国度」场地标记。

直接左移效果的来源视为施加该效果的玩家；若同一回合有多名玩家同时施加同一种全局左移效果，则按权威结算顺序最先的施加者确定来源地块。

- 同一名内卫在同一地块只保留 1 个国度标记；再次覆盖该地块不会叠加、升级或增强国度。
- 国度标记仅作为坍缩恐惧的条件判定标记，本身不产生持续效果。
- 国度标记为角色级（切走保留、切回恢复）。

## 专属技能

### 溃散

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 1 蓄力 |
| 速度 | 1 |
| 等级 | 1.5 |
| 伤害类型 | 魔法 |
| 目标 | 1 名其他存活玩家（预选） |

**结算说明**：

1. 溃散不参与技能对抗——目标使用的攻击技能对溃散不产生技能对抗效果。
2. 目标的防御技能正常抵消溃散的伤害等级。
3. 此次伤害的来源视为目标自身。因此造成的装置损失会在目标所在地块及相邻地块生成国度。

### 坍缩恐惧

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 3 气（内卫处于濒死状态时为 2 气） |
| 速度 | 1 |
| 等级 | 见结算说明 |
| 伤害类型 | 魔法 |
| 目标 | 自身顺时针及逆时针方向遇到的第一名其他存活玩家，以及所有处于国度地块上的其他存活玩家（自动） |

**结算说明**：

1. 从内卫所在地块分别向顺时针和逆时针逐格查找，对两个方向各自遇到的第一名其他存活玩家造成伤害；中间存在空地不会令该方向失去目标：
   - 内卫处于健康状态时：2 级。
   - 内卫处于濒死状态时：3 级。
2. 对所有处于国度地块上的其他存活玩家造成伤害；内卫自身即使位于自己的国度地块也不会成为目标：
   - 内卫处于健康状态时：3 级。
   - 内卫处于濒死状态时：4 级。
3. 同一玩家同时属于两侧首名目标与国度目标时只结算一次，并采用更高的国度伤害等级，不会生成两条独立伤害。

## 设计定位

内卫是阵地控制型角色。核心循环：承受伤害布置国度 → 用国度覆盖关键地块 → 坍缩恐惧对国度内玩家造成高额范围伤害。溃散绕过技能对抗，可稳定触发国度生成并对高等级对手构成威胁。装置锁定使内卫极难被单次高伤击杀——对手必须在内卫无敌回合结束后的窗口内连续施压。内卫的主要弱点是技能数量少、无治疗能力，且国度本身不提供被动收益，需要主动用坍缩恐惧兑现场地优势。
