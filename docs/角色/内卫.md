# 内卫

> 邪魔役使的国度守卫，以三枚装置替代常规生命，受伤时布置永久的国度场地，濒死时触发不破无敌自保。
>
> 通用规则（等级结算、生命结构、格挡等）见 [基础规则手册](../基础规则手册.md)。资源与 Buff 速查见 [附录](../附录/)。

## 属性

- **形态名称**：邪魔护卫
- **生命结构**：装置制（详见核心机制）
- **被动技能**：使役邪魔

## 变身收益

保留全部基础招式，额外解锁：

拳、斩、**溃散**、**坍缩恐惧**。治疗不可用。

## 核心机制：装置与国度

### 装置制生命

内卫不使用标准健康/濒死/死亡结构，改为装置计数：

- 初始拥有 3 个装置。装置计数为角色级（切走保留、切回恢复）。
- 持有 2 或 3 个装置时视为健康状态。
- 持有 1 个装置时视为濒死状态。
- 持有 0 个装置时进入死亡状态。

**伤害结算**：

结算时使用本回合收到的最大有效伤害等级判定装置损失（遵循《基础规则手册》第 4 节第 10 条）：

| 最大有效伤害等级 | 装置损失 |
|---|---|
| < 0.5 | 0 |
| ≥ 0.5 且 < 3 | 1 |
| ≥ 3 | 2 |

**装置锁定**：

若一次伤害结算导致内卫的装置数将降至 ≤ 1，则改为将装置数锁定为 1，且内卫在下回合获得「不破」效果（防御等级视为 +∞，持续 1 回合）。此效果每次从 ≥ 2 装置受击降至 ≤ 1 时均触发。

**左移效果与脆弱状态**：

草稿未明确左移健康状态效果（如剁）和脆弱状态与装置系统的交互方式。此两项列为待确认的规则缺口（见审核摘要）。

### 国度

每当内卫损失装置时，在伤害来源所在地块及该地块顺时针、逆时针各相邻一格（共 3 个地块）上各生成 1 个永久的「国度」场地标记。

- 同一地块可存在多个国度标记。
- 国度标记仅作为坍缩恐惧的条件判定标记，本身不产生持续效果。
- 国度标记为角色级（切走保留、切回恢复）。

## 专属技能

### 溃散

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 1 蓄力 |
| 速度 | 1 |
| 等级 | 1.5 |
| 伤害类型 | 魔法 |
| 目标 | 1 名其他存活玩家（预选） |

**结算说明**：

1. 溃散不参与技能对抗——目标使用的攻击技能对溃散不产生技能对抗效果。
2. 目标的防御技能正常抵消溃散的伤害等级。
3. 此次伤害的来源视为目标自身。因此造成的装置损失会在目标所在地块及相邻地块生成国度。

### 坍缩恐惧

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 3 气（内卫处于濒死状态时为 2 气） |
| 速度 | 1 |
| 等级 | 见结算说明 |
| 伤害类型 | 魔法 |
| 目标 | 自身顺时针及逆时针相邻格上的所有其他存活玩家，以及所有处于国度地块上的存活玩家（自动） |

**结算说明**：

1. 对内卫顺时针及逆时针相邻格（各 1 格）上的所有其他存活玩家造成伤害：
   - 内卫处于健康状态时：2 级。
   - 内卫处于濒死状态时：3 级。
2. 对所有处于国度地块上的存活玩家造成伤害：
   - 内卫处于健康状态时：3 级。
   - 内卫处于濒死状态时：4 级。
3. 同一玩家可同时被相邻伤害和国度伤害命中。遵循《基础规则手册》第 4 节第 10 条，健康结算仅使用最大有效伤害等级。

## 设计定位

内卫是阵地控制型角色。核心循环：承受伤害布置国度 → 用国度覆盖关键地块 → 坍缩恐惧对国度内玩家造成高额范围伤害。溃散绕过技能对抗，可稳定触发国度生成并对高等级对手构成威胁。装置锁定使内卫极难被单次高伤击杀——对手必须在内卫无敌回合结束后的窗口内连续施压。内卫的主要弱点是技能数量少、无治疗能力，且国度本身不提供被动收益，需要主动用坍缩恐惧兑现场地优势。
