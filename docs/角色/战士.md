# 战士

> 铁甲军团最后的士兵。以自身鲜血换取力量与护甲，在自伤与爆发之间建立循环。
>
> 通用规则（等级结算、生命结构、格挡等）见 [基础规则手册](../基础规则手册.md)。资源与 Buff 速查见 [附录](../附录/)。

## 属性

- **形态名称**：铁甲重装
- **生命结构**：2 HP（健康 / 濒死 / 死亡）
- **核心资源**：力量（角色级 Buff，无上限，切走保留、切回恢复）、护甲（角色级消耗型防御，切走保留、切回恢复）

## 被动技能

### 撕裂

- 战士自身健康状态每左移 1 次，获得 1 层「力量」。
- 战士使用攻击分类技能时，技能等级获得 +0.5n 的加成（n = 力量层数）。该加成通常仅用于技能对抗，不增加伤害等级；扯碎是例外，其每段伤害等级固定等于最终技能等级。
- 力量无上限。

## 变身收益

保留全部基础招式（气、蓄力、凹、紫翼双凹、剁、挡、超防、变身），额外解锁：

拳、斩、治疗、**放血**、**挑衅**、**战栗**、**熔融之拳**、**拆卸**、**欺凌**、**重振精神**、**主宰**、**血墙**、**扯碎**、**全身撞击**。

## 核心机制：护甲

护甲是战士专属的消耗型防御，以等级计量，无上限。

- **获得**：重振精神获得 2n 级护甲（n = 消耗的气量）；血墙获得 2.5 级护甲。
- **消耗**：战士受到非真实、非穿刺伤害时，在格挡结算后消耗护甲等级抵消剩余伤害等级。减免量 = min（格挡后剩余伤害等级，当前护甲等级）。伤害等级与护甲等级同时扣除相同数值。未消耗完的护甲继续保留。
- **与格挡的交互**：先结算格挡及其破碎判定，剩余伤害再消耗护甲抵消。
- **作用域**：角色级。切换角色后保留，切回时恢复。死亡时清零。

## 附加规则：易伤

易伤是施加给对手的玩家级 Debuff，层数表示剩余持续时间。

- **效果**：持有易伤的玩家受到攻击时，受到的伤害等级 +0.5。效果不随层数叠加。
- **衰减**：每回合末，持有的易伤层数 −1。包括施加易伤当回合在内的每回合末均衰减。层数归零后易伤自动移除。
- **上限**：无上限。
- **作用域**：玩家级。切换角色后保留。
- **与脆弱的叠加**：易伤与脆弱独立生效。易伤推高的伤害等级可触发脆弱即死。

## 设计定位

战士是自伤成长型爆发角色。核心循环分为两轴：

**力量轴**：放血和血墙自伤 → 触发撕裂获得力量 → 力量提升所有攻击技能的对抗等级 → 使攻击更难被技能对抗抵消。

**易伤轴**：挑衅和战栗对目标施加易伤 → 欺凌根据目标的易伤层数额外增伤 → 熔融之拳翻倍易伤层数延长持续时间 → 主宰从高易伤目标处大量获取力量。

**护甲体系**：重振精神将气转化为护甲等级；血墙以自伤代价快速获得护甲。格挡先处理来袭伤害，穿过格挡的部分再消耗护甲，使战士在自伤后仍有一定的生存缓冲。全身撞击将护甲等级兑现为伤害——护甲越高，撞击越强。

**主要风险**：

- 每次自伤消耗一个健康状态（2HP 下至多 2 次无治疗），自伤的节奏必须与治疗配合或精确计算；
- 重振精神后 2 回合内锁定非攻击技能——不能自伤、不能防御、不能治疗，对手可在此期间集火；
- 护甲被真实和穿刺伤害绕过，星神的空心拳、苦的裂空刺可直接穿透护甲；
- 全身撞击的伤害完全依赖护甲，护甲被消耗后撞击威力随之下降；
- 除扯碎外，力量仅加成技能对抗而不增加伤害；扯碎会将最终技能等级作为每段伤害等级；
- 挑衅的格挡破碎后本局不可恢复，早期使用需谨慎。
