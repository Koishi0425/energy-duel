# 凹

> 精通资源控制的熟练者，通过不断偷取资源来升级自身技能，最终将偷取行为转化为致命伤害。
>
> 通用规则（等级结算、生命结构、格挡等）见 [基础规则手册](../基础规则手册.md)。资源与 Buff 速查见 [附录](../附录/)。

## 属性

- **形态名称**：熟练形态
- **生命结构**：2 HP（健康 / 濒死 / 死亡）

## 被动技能：熟能生巧

每次使用凹、紫翼双凹或吸成功截获其他玩家本回合产出的资源时，「熟能生巧」层数 +1（最高 4 层）。气和蓄力自身产出的资源不计入；一次行动最多触发一次升级。

| 层数 | 凹与吸升级效果 |
|------|---------------|
| 1 层 | 获得 0.5 等级 |
| 2 层 | 目标选择变为后发（公开行动后再选目标） |
| 3 层 | 再加 1 等级（合计 +1.5） |
| 4 层 | 可对目标造成伤害（以当前总计等级结算） |

层数在凹凹神功使用后归零。切换角色后层数保留（角色级 Buff）。

## 变身收益

保留全部基础招式，额外解锁：

拳、斩、治疗、**吸**、**凹凹神功**。

**全局效果**：任意玩家首次变身为凹时，全场所有玩家获得「削」技能（玩家级永久 Buff）。

## 专属技能

### 吸

| 属性 | 值 |
|------|-----|
| 分类 | 资源 |
| 消耗 | 无 |
| 速度 | 1 |
| 等级 | 0（+ 熟能生巧加成） |
| 目标 | 1 名其他存活玩家（熟能生巧低于 2 层时预选；至少 2 层时后发） |

**结算说明**：类似于凹但对于蓄力的版本。若目标本回合使用蓄力，将该蓄力吸取至自身。被削无效化时失效。熟能生巧 ≥ 4 层时吸同时对目标造成对应等级的伤害。

### 凹凹神功

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 5 − n 点任意资源（n = 熟能生巧层数），由玩家在气、蓄力、辉星中自由组合 |
| 速度 | 1 |
| 等级 | 2.5 + 0.5n |
| 目标 | 1 名其他存活玩家（预选） |

**结算说明**：支付合计 (5 − 熟能生巧层数) 的任意资源，每种资源只能选择整数点数，可以混合气、蓄力及当前持有的其他特殊资源；李淳罡斩所产生的 `1/3`、`2/3` 余额不能单独计作 1 点支付。造成 2.5 + 0.5×层数的攻击，使用后熟能生巧层数归零。

| 层数 | 消耗 | 等级 |
|------|------|------|
| 0 | 5 点 | 2.5 |
| 1 | 4 点 | 3.0 |
| 2 | 3 点 | 3.5 |
| 3 | 2 点 | 4.0 |
| 4 | 1 点 | 4.5 |

满层时只需支付 1 点任意资源即可造成 4.5 级攻击——全游戏最高基础伤害等级。

### 削

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 无 |
| 速度 | 1 |
| 等级 | 0 |
| 目标 | 本回合使用吸的所有玩家（自动） |
| 获得方式 | 任意玩家变身为凹时，所有玩家自动获得此技能（玩家级永久 Buff） |

**结算说明**：取消本回合全场所有吸，并使吸的使用者损失一个健康状态。与剁对凹的克制关系完全对称。结算在回合最优先阶段处理。

## 设计定位

凹是一个滚雪球型角色。早期通过凹和吸干扰对手资源获取并积累层数，中期在层数支撑下获得后发选择和信息优势，后期释放凹凹神功终结比赛。全场削的存在迫使所有吸使用者彼此制约——一旦有人放了削，所有吸使用者都会受罚。凹凹神功的任意资源支付提供了极大的灵活性。
