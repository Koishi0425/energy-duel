# 苦

> 千锤百炼的武道苦修者，以预判对手行动类型为核心，成功应对后永久成长，逐步解锁更高等级的反制技能。
>
> 通用规则（等级结算、生命结构、格挡等）见 [基础规则手册](../基础规则手册.md)。资源与 Buff 速查见 [附录](../附录/)。

## 属性

- **形态名称**：苦修形态
- **生命结构**：2 HP（健康 / 濒死 / 死亡）

## 被动技能

### 千锤百炼

苦的所有招式（含基础招式）在结算时会对目标行动进行判定。若因苦的招式导致其他玩家的行动未达到预期效果，视为**应对成功**。每次应对成功获得 0.5 气，并使「千锤百炼」Buff 层数 +1（角色级，至多 4 层，切走保留、切回恢复）。

以下为常见应对成功场景（非穷举——结算时以"目标行动预期效果是否被苦的招式阻止"为准）：

- **凹 / 紫翼双凹**：成功偷取目标的气或蓄力（目标预期获得资源但被夺走）。
- **剁**：成功取消其他玩家的凹类技能。
- **挡 / 超防**：完全抵消了至少一次攻击（攻击方预期造成伤害但被格挡）。
- **拳 / 斩**：在目标行动结算前将目标击杀（目标行动被跳过）。
- **杖责**：使目标该回合所获资源无效。
- **挪移**：将目标对苦的攻击转移至其他玩家。
- **看破**：打断（取消）目标的攻击或状态类行动。
- **崩裂**：使目标的防御被击穿并附加额外效果。

「千锤百炼」Buff 层数以 n 表示（0 ≤ n ≤ 4），用于计算各专属技能的等级和速度加成。

## 变身收益

保留全部基础招式，额外解锁：

拳、斩、治疗、**裂空刺**、**杖责**、**挪移**、**看破**、**崩裂**。

## 专属技能

### 裂空刺

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 2 蓄力 |
| 速度 | n（至多为 4） |
| 等级 | n（至多为 4） |
| 伤害类型 | 真实 |
| 目标 | 1 名其他存活玩家（预选） |

**结算说明**：

1. 以当前 n 值作为等级和速度。
2. 真实伤害结算时仍受目标格挡影响，但无视神体与护甲；仍受基础规则限制：等级低于 3 时最多左移 1 层健康状态。
3. 使用裂空刺不消耗 n。

### 杖责

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 1 气 |
| 速度 | 1 + 0.5n（向上取整） |
| 等级 | 1.5 |
| 伤害类型 | 钝击 |
| 目标 | 1 名其他存活玩家（预选） |

**结算说明**：

1. 对所选玩家造成 1.5 级钝击伤害。
2. 若该玩家本回合使用了资源分类招式（如气、蓄力、隐秘藏品等），则该玩家本回合从该招式获得的所有资源无效——视为应对成功。
3. 若该玩家未使用资源分类招式，杖责仍造成伤害，但不算应对成功。

### 挪移

| 属性 | 值 |
|------|-----|
| 分类 | 防御 |
| 消耗 | 无 |
| 速度 | 1 + 0.5n（向上取整） |
| 等级 | 0.5 + n（格挡） |
| 目标 | 1 名其他存活玩家（预选） |

**结算说明**：

1. 提供 0.5 + n 级格挡，对所有指向苦的攻击生效。
2. 若所选玩家本回合对苦使用了攻击分类招式，选择其中等级最高的一次攻击，将其目标改为顺时针方向的下一位存活玩家（跳过苦自身）。转移后的攻击由新目标重新进行防御结算——视为应对成功。
3. 当场中仅剩 2 名存活玩家时，转移目标改为原攻击者自身（攻击者自己承受此次攻击）。
4. 若所选玩家本回合未攻击苦，挪移仍提供格挡，但不算应对成功。

### 看破

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 1 蓄力 |
| 速度 | 1 + 0.5n（向上取整） |
| 等级 | 0.5（未应对）/ 0.5 + n（应对成功时） |
| 伤害类型 | 钝击 |
| 目标 | 1 名其他存活玩家（预选） |

**结算说明**：

1. 若所选玩家本回合使用了攻击分类或特殊分类招式：
   - 取消该玩家的该行动效果（该行动不产生任何作用）。
   - 看破等级按 0.5 + n 计算，对该玩家造成伤害——视为应对成功。
2. 若所选玩家未使用攻击或特殊分类招式，看破仅以 0.5 级造成伤害，不算应对成功。

### 崩裂

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 1 气 |
| 速度 | 1 + n |
| 等级 | 1 + n |
| 伤害类型 | 钝击 |
| 目标 | 1 名其他存活玩家（预选） |

**结算说明**：

1. 对所选玩家造成 1 + n 级钝击伤害。
2. 若该玩家本回合使用了防御分类招式：
   - 该玩家受到此次攻击时，崩裂的等级额外 + 0.5n（即总等级 = 1 + n + 0.5n = 1 + 1.5n）。
   - 该玩家获得「崩裂禁防」Debuff（玩家级，持续 2 回合）：不能提交防御分类招式——视为应对成功。
3. 若该玩家未使用防御分类招式，崩裂仍造成伤害，但不算应对成功且不附加禁防。

## 设计定位

苦是预判型反制角色。核心循环：使用专属技能预判对手行动类型 → 应对成功积累 n → 更高的 n 使技能等级和速度提升 → 继续压制。裂空刺作为真实伤害终结技，可穿透一切防御但受 n 限制且消耗 2 蓄力。主要风险在于预判失败时技能效果大打折扣（看破仅 0.5 级、杖责和崩裂无附加效果），且 n = 0 时速度普遍偏低（仅 1），容易被更快的对手抢先结算。反制窗口在于苦的预判——若对手变换行动类型可让苦的应对落空。
