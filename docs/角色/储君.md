# 储君

> 群星王座的继承人。拥有宇宙的力量，但总是让他的仆从们去做各种事情。
>
> 通用规则（等级结算、生命结构、格挡等）见 [基础规则手册](../基础规则手册.md)。资源与 Buff 速查见 [附录](../附录/)。

## 属性

- **形态名称**：辉星王储
- **生命结构**：2 HP（健康 / 濒死 / 死亡）
- **被动技能**：无（辉星机制由变身结算触发）

## 变身收益

保留全部基础招式，额外解锁：

拳、斩、治疗，以及 10 个辉星/君王之剑体系技能。

## 核心机制：辉星与君王之剑

### 辉星

辉星是储君专属的第三类资源（见 [资源一览](../附录/资源一览.md)），用于驱动绝大部分技能：

- **持久性**：辉星不因回合结束或切换角色而消失。切到其他角色后辉星数值保留，切回储君时恢复。
- **首次变身**：本局第一次变身为储君时立即获得 3 辉星。该效果为玩家级（「辉星君权」Buff），切走再切回不会重复触发。

### 君王之剑

由锻造等级和激活状态组成，均为角色级（切走保留、切回恢复）：

- **锻造等级**：通过铸剑者、筑墙和征召上前提升，最高 3 级，支持 0.5 级增量。
- **激活状态**：首次锻造（从 0 提升到 > 0）时自动激活。君王之剑使用后激活状态被移除（即锁定），继续锻造不会解除锁定——必须使用征召上前重新激活。

## 专属技能

### 收集光辉

| 属性 | 值 |
|------|-----|
| 分类 | 防御 |
| 消耗 | 无 |
| 速度 | 2 |
| 等级 | 2（格挡） |
| 目标 | 自身（自动） |

**结算说明**：提供 2 级格挡。承受等级 ≥ 2 的攻击后破碎，之后收集光辉等级为 0。若本回合完全抵消了至少一次攻击，且在结算全程没有受到任何伤害（HP 无变化），则在回合结束时获得 1 辉星。

### 流光溢彩

| 属性 | 值 |
|------|-----|
| 分类 | 防御 |
| 消耗 | 1 气 |
| 速度 | 2 |
| 等级 | 3（格挡） |
| 目标 | 自身（自动） |

**结算说明**：生成 3 级临时格挡。此外获得「流光余辉」Buff（角色级，持续至下回合结算）：下回合受到攻击时自身招式等级最低视为 1.5。若实际所选招式等级高于 1.5，则使用原等级。

### 隐秘藏品

| 属性 | 值 |
|------|-----|
| 分类 | 资源 |
| 消耗 | 1 气 |
| 速度 | 2 |
| 等级 | 0 |
| 目标 | 自身（自动） |

**结算说明**：立即获得 1 辉星，并获得「隐秘藏品」Buff（角色级）。下回合开始时自动获得 3 辉星并移除该 Buff。配合首次变身的 3 辉星，储君可在两回合内积累足够的启动资金。

### 粒子墙

| 属性 | 值 |
|------|-----|
| 分类 | 防御 |
| 消耗 | 2n 辉星（n 为正整数） |
| 速度 | 2 |
| 等级 | 2n（格挡） |
| 目标 | 自身（自动） |

**结算说明**：提交时选择正整数 n，支付 2n 辉星，生成 2n 级临时格挡。破碎仅影响本次生成的粒子墙。

### 胜券在王

| 属性 | 值 |
|------|-----|
| 分类 | 资源 |
| 消耗 | 5 辉星 |
| 速度 | 2 |
| 等级 | 0 |
| 目标 | 自身（自动） |

**结算说明**：支付 5 辉星，获得 9 辉星（净增 4 辉星）。出招期间处于脆弱状态。建议在有防御保障时使用，否则极易被击杀。

### 星尘

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 当前持有的全部辉星（至少 1） |
| 速度 | 2 |
| 等级 | 单个 0.5 级，总计 n/2 级 |
| 目标 | 其他存活玩家，分配 n 次（后发，可重复） |

**结算说明**：提交时由服务端将 n 固定为当前持有的全部辉星并全部支付，玩家不能保留一部分辉星。所有存活玩家提交后，服务端公开本回合全部行动，然后要求星尘使用者向其他存活玩家分配 n 个 0.5 级攻击。可对同一名玩家重复分配——例如持有 4 辉星时会全部消耗，并可将 4 个 0.5 级全部分配给同一名玩家（总计 2 级），或分散给多名玩家。

### 铸剑者

| 属性 | 值 |
|------|-----|
| 分类 | 资源 |
| 消耗 | 1 气 + 4 辉星 |
| 速度 | 2 |
| 等级 | 0 |
| 目标 | 自身（自动） |

**结算说明**：君王之剑锻造等级 +3。若此前锻造等级为 0（即首次锻造），自动激活君王之剑。用于快速将君王之剑拉升等级。

### 筑墙

| 属性 | 值 |
|------|-----|
| 分类 | 防御 |
| 消耗 | 1 气 |
| 速度 | 2 |
| 等级 | 2.5（格挡） |
| 目标 | 自身（自动） |

**结算说明**：生成 2.5 级临时格挡，同时锻造 +1（首次锻造时自动激活）。

### 君王之剑

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 无 |
| 速度 | 2 |
| 等级 | 当前锻造等级 |
| 目标 | 1 名其他存活玩家（预选） |
| 解锁条件 | 君王之剑已生成（锻造 > 0）且处于激活状态 |

**结算说明**：以当前锻造等级攻击所选玩家。使用后君王之剑进入锁定状态。注意：继续锻造不会解除锁定，必须使用征召上前重新激活。

### 征召上前

| 属性 | 值 |
|------|-----|
| 分类 | 特殊 |
| 消耗 | 1 气 |
| 速度 | 2 |
| 等级 | 0 |
| 目标 | 自身（自动） |
| 解锁条件 | 无，始终可用 |

**结算说明**：先令锻造等级 +0.5。若君王之剑尚未生成，则此次锻造会生成它；随后无论此前是否已经生成或激活，均将君王之剑重置为可使用的激活状态。因此它既可用于从零建立君王之剑，也可在王剑使用后重新激活，且王剑已经激活时仍然可以使用。

## 设计定位

储君是全游戏最复杂的经济型角色。核心循环：积累辉星 → 锻造 → 防御/增值 → 释放君王之剑 → 征召重新激活。星尘的后发机制使他成为唯一可以在看到所有行动后再决定打击目标的角色。他的主要弱点是技能消耗大量辉星，且出招节奏需要多个回合的积累和规划。
