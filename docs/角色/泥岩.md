# 泥岩

> 沃土重装的守护者，以拳势的永久成长和屏障系统构筑防线，并通过沉睡换取斩击能力。
>
> 通用规则（等级结算、生命结构、格挡等）见 [基础规则手册](../基础规则手册.md)。资源与 Buff 速查见 [附录](../附录/)。

## 属性

- **形态名称**：沃土重装
- **生命结构**：2 HP（健康 / 濒死 / 死亡）

## 被动技能

### 土石之子

- 技能树中无斩（初始无法使用斩——需通过秽壤的血脉苏醒后解锁）。
- 拳不再脆弱——但使用拳时若受到等级差 ≥ 0.5 的伤害（即拳未能抵消对方攻击等级），直接死亡。若因防御技能格挡成功而未被伤害，则不触发此即死。
- 每次使用拳，「锤炼拳势」Buff 层数 +1，每层使拳的等级永久提高 0.5。该效果跨回合累积，切走角色保留。

### 沃土予身

- 变身当回合开始计时（计数器从变身回合开始累加）。
- 无屏障时每回合计数器 +1（使用变身时当回合不计入）。
- 计数器满 4 时：获得一层「屏障」Buff（上限 1 层），计数器归零。
- 屏障效果：抵消一次未被防御的伤害（任意等级），同时自身恢复一个健康状态（右移一层）。屏障失去后计数器从 0 重新开始。

## 变身收益

保留全部基础招式（斩除外），额外解锁：

拳（非脆弱版，但附带土石之子的风险）、治疗、**岩崩锤**、**秽壤的血脉**。通过沉睡苏醒获得斩，沉睡期间可使用「继续沉睡」。

## 专属技能

### 岩崩锤

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 无 |
| 速度 | 1（攻击）；4（自愈） |
| 等级 | 2 |
| 伤害类型 | 钝击 |
| 目标 | 自身格及顺、逆时针各 2 格内的所有其他存活玩家（自动） |
| 解锁条件 | 持有至少 1 层「岩崩锤次数」Buff |

**次数获取**：每次泥岩被其他玩家明确选择为目标（凹、吸、定向攻击等），岩崩锤计数器 +1。自动范围效果、路径判定和地块选择不属于“选择泥岩”，不会计数。每满 3 次自动消耗并转化为 1 层「岩崩锤次数」（可累积多层，无上限）。

**结算说明**：

1. **先自愈**（预处理阶段，速度 4）：自身恢复一个健康状态（右移一层）。
2. **再攻击**（速度 1）：对自身格及顺、逆时针最多各两格（共 5 格）内的所有其他存活玩家造成 2 级钝击伤害。
3. 消耗 1 层使用次数。

### 秽壤的血脉

| 属性 | 值 |
|------|-----|
| 分类 | 特殊 |
| 消耗 | 3 气 |
| 速度 | 0 |
| 等级 | 0 |
| 目标 | 自身（自动） |

**结算说明**：

1. **进入沉睡**：获得「沉睡」Buff（2 层）和「沉睡积蓄」Buff（1 层）。沉睡期间：不能被其他玩家选择为目标；同时免疫一切伤害，因此自动范围与路径伤害也无法伤害泥岩。
2. **沉睡期间的选择**：
   - 提交「继续沉睡」（由沉睡 Buff 临时授予）：消耗 1 层沉睡层数，积蓄 +1。当沉睡层数归零时自动苏醒。
   - 提交任意其他行动：提前苏醒。未消耗的沉睡层数对应的气全额返还（每层 1 气）。
3. **苏醒效果**：移除沉睡和积蓄 Buff；永久获得斩（「血脉斩」Buff 授予 slash 技能）；获得「血脉苏醒」Buff（层数 = 已沉睡回合数含当前回合，持续等长回合数），每层使斩的等级 +0.5。

### 继续沉睡

| 属性 | 值 |
|------|-----|
| 分类 | 特殊 |
| 消耗 | 无 |
| 速度 | 0 |
| 等级 | 0 |
| 目标 | 自身（自动） |
| 获得方式 | 沉睡 Buff 临时授予 |

**结算说明**：保持沉睡一回合。沉睡层数 −1，积蓄层数 +1。若层数归零则自动苏醒。第三回合结束后必苏醒。

## 设计定位

泥岩是防御型成长角色。早期靠屏障系统自保并积攒岩崩锤次数。中期通过反复使用拳永久提升拳的等级——最终拳可以达到远超普通斩击的等级，但土石之子用拳自损的高风险使其无法无脑使用。后期用秽壤血脉沉睡解锁斩，获得大幅度攻击加成。典型的泥岩路线：积攒屏障 → 积攒锤次数 → 低风险用拳 → 沉睡解锁斩 → 高攻收尾。沃土予身的屏障抵消任意伤害加自愈的组合让泥岩成为最难被一套击杀的角色，但屏障只能存一层且每四回合才生成一层，对手可以在屏障空窗期发动猛攻。
