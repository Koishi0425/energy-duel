# 娇斯拉

> 原子吐息的驾驭者，以高昂的资源消耗换取无条件的 3 级定向攻击。
>
> 通用规则（等级结算、生命结构、格挡等）见 [基础规则手册](../基础规则手册.md)。资源与 Buff 速查见 [附录](../附录/)。

## 属性

- **形态名称**：原子形态
- **生命结构**：2 HP（健康 / 濒死 / 死亡）
- **被动技能**：无

## 变身收益

保留全部基础招式（气、蓄力、凹、紫翼双凹、剁、挡、超防、变身），额外解锁：

拳、斩、治疗、**原子吐息**。

## 专属技能

### 原子吐息

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 2 气 + 1 蓄力 |
| 速度 | 1 |
| 等级 | 3 |
| 目标 | 1 名其他存活玩家（预选） |

**结算说明**：对所选玩家发动 3 级攻击。3 级攻击可突破「最多左移一层状态」的限制——当等级差 ≥ 1 时可直接从健康状态击杀变身后的角色。

## 设计定位

娇斯拉的技能树简洁克制，只有一个高消耗大招。她的策略核心在于资源管理——如何在积累 2 气 1 蓄力的过程中不被凹打断，并在合适的时机释放以终结比赛。3 级原子吐息是无条件最高等级的定向攻击，但其 2 气 1 蓄力的消耗也是全游戏除鬼影重重外最高的。
