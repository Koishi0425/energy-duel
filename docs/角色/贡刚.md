# 贡刚

> 战斧的持有者，通过举斧激活强化状态，使斩击更具威胁并以斧挡建立攻防一体的节奏。
>
> 通用规则（等级结算、生命结构、格挡等）见 [基础规则手册](../基础规则手册.md)。资源与 Buff 速查见 [附录](../附录/)。

## 属性

- **形态名称**：战斧形态
- **生命结构**：2 HP（健康 / 濒死 / 死亡）
- **被动技能**：无

## 变身收益

保留全部基础招式，额外解锁：

拳、斩、治疗、**举斧**、**斧挡**（条件解锁）。

## 专属技能

### 举斧

| 属性 | 值 |
|------|-----|
| 分类 | 特殊 |
| 消耗 | 2 气 |
| 速度 | 1 |
| 等级 | 0 |
| 目标 | 自身（自动） |

**结算说明**：获得「举斧」Buff（角色级，无持续时间限制）。举斧状态下：
- 解锁斧挡技能；
- 斩的等级提高 0.5（从 1.5 变为 2.0）。

举斧 Buff 在切走角色后保留，切回时自动恢复。

### 斧挡

| 属性 | 值 |
|------|-----|
| 分类 | 防御 |
| 消耗 | 无 |
| 速度 | 2 |
| 等级 | 2.5（格挡） |
| 目标 | 自身（自动） |
| 解锁条件 | 需持有「举斧」Buff |

**结算说明**：提供 2.5 级格挡。承受等级 ≥ 2.5 的攻击后破碎并返还 1 气，之后斧挡等级为 0。

## 设计定位

贡刚的节奏分两段——先用 2 气举斧建立状态，之后斩变为 2.0 级、斧挡可免费使用并提供高等级防御。斧挡破碎后等级为 0。
