# 李淳罡

> 无法使用拳的纯粹剑客，通过被动将斩的消耗降至极低，并以后发斩击在公开行动后精准选择目标。
>
> 通用规则（等级结算、生命结构、格挡等）见 [基础规则手册](../基础规则手册.md)。资源与 Buff 速查见 [附录](../附录/)。

## 属性

- **形态名称**：剑道宗师
- **生命结构**：2 HP（健康 / 濒死 / 死亡）

## 被动技能：剑道

- 无法使用拳（拳不在技能树中）。
- 斩的消耗从 1 气降为 1/3 气。即正常斩消耗 1 气，李淳罡只需 1/3 气。

## 变身收益

保留全部基础招式（拳除外），额外解锁：

斩（1/3 气消耗）、治疗、**剑气**、**一剑开天门**。

## 核心机制：后发斩击

剑气和一剑开天门的目标选择时机为「后发」：在选择阶段不指定目标，所有玩家提交后公开行动，再选择目标。与储君的星尘不同，李淳罡的后发是分配一个完整的攻击（而非拆分），且分配完毕后不公开给其他后发等待者。

## 专属技能

### 剑气

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 1 气 |
| 速度 | 1 |
| 等级 | 1.5 |
| 伤害类型 | 斩击 |
| 目标 | 1 名其他存活玩家（后发） |

**结算说明**：行动公开后选择目标，造成 1.5 级斩击。后发优势在于可以避开超防和格挡。

### 一剑开天门

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 2 气 + 2 蓄力 |
| 速度 | 1 |
| 等级 | 3 |
| 伤害类型 | 斩击 |
| 目标 | 1 名其他存活玩家（后发） |

**结算说明**：与剑气相同的后发机制，以 3 级发动斩击。3 级可突破状态限制直接致死。

## 设计定位

李淳罡是纯粹的攻击手。1/3 气的斩使他几乎可以每回合使用斩（3 次斩仅耗 1 气）。两个后发大招可以绕过对手的防御——看到谁放了超防就换人攻击。代价是完全无法使用拳，也没有任何范围和特殊机制。伤害类型「斩击」用于未来可能的属性克制系统。
