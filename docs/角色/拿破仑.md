# 拿破仑

> 法兰西的皇帝，以炮兵指挥官的身份降临战场。无法使用任何基础资源，仅凭三种指令的排列组合驾驭战局——短的战术配合稳扎稳打，长的传奇战役足以改写历史。
>
> 通用规则（等级结算、生命结构、格挡等）见 [基础规则手册](../基础规则手册.md)。资源与 Buff 速查见 [附录](../附录/)。

## 属性

- **形态名称**：皇帝形态
- **生命结构**：2 HP（健康 / 濒死 / 死亡）
- **资源限制**：拿破仑不能获得或消耗气与蓄力。任何产生气或蓄力的效果对拿破仑无效，任何消耗气或蓄力的技能拿破仑无法使用。凹和紫翼双凹对拿破仑无实际收益。

## 核心机制：指挥系统

拿破仑变身后，所有通用招式（气、蓄力、凹、紫翼双凹、剁、挡、超防、拳、斩、治疗、变身）均被禁用。仅保留三种**指令**。

### 指令缓冲

拿破仑拥有一个指令缓冲（角色级，至多 6 条，切走角色后清空）。缓冲中的指令有两个作用：

1. **序列匹配**：缓冲中连续的子序列可以组成策略。玩家在提交指令时可以择机执行策略，消耗对应指令、触发策略效果。
2. **战术优势引擎**：每回合开始时，缓冲中每有一条战术指令（T），拿破仑获得 1 层「战术优势」。

### 战术优势

- 角色级 Buff，至多 6 层。切走角色时清空。
- **每层效果**：拿破仑所有指令效果和策略效果的等级 +0.5。
- **来源**：
  - 回合开始时，缓冲中每个 T 产生 1 层；
  - 永久 Buff「皇帝」（来自 TTTTT）每回合开始时额外产生 1 层；
  - 永久 Buff「不落要塞」（来自 DDDDD）持续期间每回合开始时额外产生 1 层。
- **消耗**：部分策略标注"消耗全部战术优势"，执行后层数归零，策略等级按消耗前的层数计算。
- **清空**：切换角色时战术优势层数归零（缓冲清空导致 T 引擎消失）。「神机」「皇帝」「百日王朝」等永久 Buff 保留，切回拿破仑后重新开始产出战术优势。

### 回合流程

拿破仑每回合的行动选择分为两种模式：

**A) 发布新指令**：提交一个指令（A / D / T）。该指令的基础效果在本回合结算，同时指令追加到缓冲末尾（超出 6 条时最旧的被移除）。若追加后缓冲中存在至少一个可识别的策略序列，玩家可以选择是否执行该策略——若执行，策略效果**替代**本次指令的基础效果，消耗的连续指令从缓冲中移除。若不执行，缓冲维持不变。

**B) 纯执行策略**：不发布新指令，直接消耗缓冲中已有的一个连续子序列来执行对应策略。本回合不向缓冲追加任何指令。

若缓冲中没有任何可识别的策略序列，则只能选择 A。

### 策略序列规则

- 策略由缓冲中**连续**的指令子序列匹配。同一缓冲中可能匹配到多个策略（不同位置或不同长度），玩家选择其中一个执行。
- 执行后，消耗的指令从缓冲中移除，剩余指令保持原有顺序。
- 单条指令（A / D / T）是发布指令时的基础效果，不作为可从缓冲中单独执行的策略；正式策略最短为 2 条指令。
- 同一个指令不能被多个策略重复消耗——执行后即从缓冲移除。

## 变身收益

保留：无（全部禁用）。

变身为拿破仑时，变身技能同时被禁用。拿破仑必须通过完成「厄尔巴逃逸」策略才能永久解锁变身，之后可以像其他角色一样自由切换。

## 专属技能

拿破仑只有三个指令，均无资源消耗。

### 攻击指令（A）

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 无 |
| 速度 | 1 |
| 等级 | 1 |
| 伤害类型 | 钝击 |
| 目标 | 1 名其他存活玩家（预选） |

**基础效果**：对所选玩家造成 1 级钝击伤害。

### 防守指令（D）

| 属性 | 值 |
|------|-----|
| 分类 | 防御 |
| 消耗 | 无 |
| 速度 | 1 |
| 等级 | 1.5（格挡） |
| 目标 | 自身（自动） |

**基础效果**：提供 1.5 级格挡（重新生成型——每次施放独立计算，不持久破碎）。

### 战术指令（T）

| 属性 | 值 |
|------|-----|
| 分类 | 特殊 |
| 消耗 | 无 |
| 速度 | 1 |
| 等级 | 0 |
| 目标 | 自身（自动） |

**基础效果**：指令追加到缓冲。无直接战斗效果。

T 的核心价值在于：（1）在缓冲中持续产生战术优势；（2）作为策略序列的必要组成；（3）与其他指令排列组合触发高级策略。

所有策略产生的格挡均为重新生成型。承受不低于当前防御等级的攻击时，该格挡当次破碎。之后再次执行同一策略时，重新生成完整格挡。

## 策略序列表

> 下表所列等级均为**基础等级**。执行时，实际等级 = 基础等级 + 当前战术优势层数 × 0.5（每层 +0.5 级）。

### 长度 1：基础指令

始终可用，作为保底。

| 序列 | 名称 | 效果 | 速度 |
|------|------|------|------|
| A | 炮击 | 1 级钝击，1 名其他存活玩家 | 1 |
| D | 布防 | 1.5 级格挡 | 1 |
| T | 谋划 | 指令存入缓冲（不产生即时战斗效果；回合开始时缓冲中的 T 已产生战术优势） | 1 |

### 序列学习法则

玩家无需背诵全部序列表。理解以下三条法则即可推断绝大多数序列的效果：

**法则一：重复强化**——同一指令连续重复，每多一次效果显著增强。A 序列提升攻击等级，D 序列提升格挡等级，T 序列每多一个多产 1 层战术优势并在特定长度解锁永久 Buff。

**法则二：两两配合**——任意两种不同指令的组合都有独一无二的身份。攻击+防守 = 稳步推进；防守+攻击 = 盾反；攻击+战术 = 校准标记；战术+攻击 = 优势放大；防守+战术 = 防御中积蓄；战术+防守 = 优势布防。

**法则三：三段定式**——三条指令的序列中，T 的位置决定策略性质。T 在首位 = 战术准备后放大后续行动；T 在末位 = 行动后获得下回合的增益（速度或等级）；T 在中间 = 行动中穿插战术调整产生控制效果。纯粹的三连 A / 三连 D / 三连 T 则是法则一的延伸。

## 特殊交互

| 场景 | 处理 |
|------|------|
| 凹对拿破仑 | 拿破仑无气可偷，凹无收益 |
| 剁对拿破仑 | 拿破仑的指令不属于凹类技能，剁不触发 |
| 恐惧对拿破仑 | 拿破仑没有气技能，恐惧期间所有指令仍可使用（恐惧"只能使用气"对拿破仑放宽为"可正常使用指令"） |
| 拿破仑切换角色 | 指令缓冲清空，战术优势层数归零。「神机」「皇帝」「百日王朝」「不落要塞」等永久 Buff 切走保留、切回恢复 |
| 变身被禁用期间 | 拿破仑无法切换为其他角色，直到完成「厄尔巴逃逸」 |

## 设计定位

拿破仑是全游戏最特殊的角色——他完全脱离气/蓄力经济，以时间为唯一资源。核心循环为：发布指令填充缓冲 → 缓冲中的 T 每回合产出战术优势 → 择机消耗缓冲中的连续子序列执行策略 → 策略效果随战术优势层数和序列长度双重放大。

**三阶段节奏**：

1. **开局（积攒期）**：频繁使用 T 和 D 填充缓冲。缓冲中的 T 开始每回合产出战术优势，D 保障生存。此时攻击力有限，以防守和积累为主。
2. **中期（爆发期）**：缓冲填至 3–5 条后，开始择机执行长度 3–4 的策略。TTA 消耗全部优势打出高额单体伤害，AADD 在攻击的同时保持防御。缓冲的灵活管理允许在一局中多次执行不同策略。
3. **后期（传奇期）**：若能存活至缓冲积累 4–5 条，可执行长度 4–5 的传奇策略——AAAAA 范围毁灭、TTTTT 永久皇帝引擎、TATAT 解锁变身获取退路。长度 5 的策略具有终结比赛的能力。

**优势**：零资源消耗，战术极度灵活，可在攻/防/增益之间自由切换；战术优势的累积使后期指令威力远超同类免费技能。

**风险**：前期单个指令强度偏低（A 仅 1 级，D 仅 1.5 级），在战术优势积累起来之前极易被压制；策略执行需消耗缓冲中的指令，过度执行会导致缓冲枯竭、战术优势断档；所有行动速度均为 1–2（仅少数长度 5+ 策略达到速度 3），面对速度 3–4 的对手可能被抢先。

**反制窗口**：拿破仑在缓冲空旷（0–2 条）时最为脆弱——此时可执行的策略少、战术优势产出低。对手应在其积攒缓冲的早期回合集中施压。此外，拿破仑的策略序列完全透明（对手可从其历史指令推断缓冲内容），有经验的对手可以预判其下一手并提前应对。
