# 拿破仑

> 法兰西的皇帝，以炮兵指挥官的身份降临战场。无法使用任何基础资源，仅凭三种指令的排列组合驾驭战局——短的战术配合稳扎稳打，长的传奇战役足以改写历史。
>
> 通用规则（等级结算、生命结构、格挡等）见 [基础规则手册](../基础规则手册.md)。资源与 Buff 速查见 [附录](../附录/)。

## 属性

- **形态名称**：皇帝形态
- **生命结构**：2 HP（健康 / 濒死 / 死亡）
- **资源限制**：拿破仑不能获得或消耗气与蓄力。任何产生气或蓄力的效果对拿破仑无效，任何消耗气或蓄力的技能拿破仑无法使用。凹和紫翼双凹对拿破仑无实际收益。

## 核心机制：指挥系统

拿破仑变身后，所有通用招式（气、蓄力、凹、紫翼双凹、剁、挡、超防、拳、斩、治疗、变身）均被禁用。仅保留三种**指令**。

### 指令缓冲

拿破仑拥有一个指令缓冲（角色级，至多 6 条，切走角色后清空）。缓冲中的指令有两个作用：

1. **序列匹配**：缓冲中连续的子序列可以组成策略。玩家可以择机执行策略，消耗对应指令、触发策略效果。
2. **策略唯一来源**：拿破仑的所有增益均来自主动执行策略——单纯持有指令不会被动产生任何战斗力提升。

### 战术优势

- 角色级 Buff，至多 6 层。切走角色时清空。
- **每层效果**：拿破仑所有指令效果和策略效果的等级 +0.5（攻击同时提高技能等级和伤害等级，防御提高格挡等级；不影响速度）。
- **来源**：执行以下策略后获得，层数叠加；重复获得时持续时间保留当前值与新来源值中的较大者：
  - TT（战略谋划）：2 层，持续 3 回合；
  - TTT（皇帝诏令）：3 层，持续 3 回合；
  - TTTT（神机妙算）：4 层，持续 4 回合；
  - TTTTT（加冕称帝）：5 层，持续 5 回合；
  - DT（防御部署）：1 层，持续 3 回合；
  - AT（校准射击）：1 层，持续 3 回合。
- **引擎 Buff**：以下 Buff 每回合开始时自动产出战术优势（层数叠加至上限 6，不单独计时——引擎 Buff 到期后对应产出停止）：
  - 皇帝：每回合 +2 层，持续 5 回合；
  - 百日王朝：每回合 +1 层，永久；
  - 不落要塞：每回合 +1 层，持续 3 回合。
- **消耗**：TTA（雷霆一击）和 TTAA（大帝亲征）标注"消耗全部战术优势"，执行后层数归零。
- **清空**：切换角色时战术优势层数归零（缓冲清空且引擎 Buff 切走后不再产出）。神机、皇帝、百日王朝、不落要塞等 Buff 切走保留、切回恢复。

### 回合流程

拿破仑每回合的行动由面板选择直接决定，没有额外确认窗口：

1. **执行已有策略**：策略卡会显示缓冲中已有的连续子序列；点击后直接执行，不追加指令。
2. **指令触发策略**：策略卡也会显示“缓冲尾部补上本回合末位指令后”可组成的策略。点击后自动补上该末位指令并执行最长的尾部策略；该指令不会产生基础效果，且连同尾部序列一并从缓冲移除。
3. **发布普通指令**：点击 A / D / T 始终正常结算所选指令的基础效果，并将它追加到缓冲末尾（超出 6 条时移除最旧指令）；即使该指令能组成策略，也不会自动替换为策略。

当同一策略既已存在于缓冲、又可能由新指令组成时，优先执行已有缓冲，绝不会额外追加指令。

### 策略序列规则

- 策略由缓冲中**连续**的指令子序列匹配。同一缓冲中可能匹配到多个策略（不同位置或不同长度），玩家选择其中一个执行。
- 执行后，消耗的指令从缓冲中移除，剩余指令保持原有顺序。
- 单条指令（A / D / T）是发布指令时的基础效果，不作为可从缓冲中单独执行的策略；正式策略最短为 2 条指令。
- 同一个指令不能被多个策略重复消耗——执行后即从缓冲移除。

## 变身收益

保留：无（全部禁用）。

变身为拿破仑时，变身技能同时被禁用。拿破仑必须通过完成「厄尔巴逃逸」策略才能永久解锁变身，之后可以像其他角色一样自由切换。

## 专属技能

拿破仑只有三个指令，均无资源消耗。

### 攻击指令（A）

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 无 |
| 速度 | 1 |
| 等级 | 1 |
| 伤害类型 | 钝击 |
| 目标 | 1 名其他存活玩家（预选） |

**基础效果**：对所选玩家造成 1 级钝击伤害。

### 防守指令（D）

| 属性 | 值 |
|------|-----|
| 分类 | 防御 |
| 消耗 | 无 |
| 速度 | 1 |
| 等级 | 1.5（格挡） |
| 目标 | 自身（自动） |

**基础效果**：提供 1.5 级格挡（重新生成型——每次施放独立计算，不持久破碎）。

### 战术指令（T）

| 属性 | 值 |
|------|-----|
| 分类 | 特殊 |
| 消耗 | 无 |
| 速度 | 1 |
| 等级 | 0 |
| 目标 | 自身（自动） |

**基础效果**：指令追加到缓冲。无直接战斗效果。

T 的核心价值在于：（1）作为策略序列的必要组成；（2）与其他指令排列组合触发策略后获得战术优势。

所有策略产生的格挡均为重新生成型。承受不低于当前防御等级的攻击时，该格挡当次破碎。之后再次执行同一策略时，重新生成完整格挡。

## 策略序列表

> 下表所列等级均为**基础等级**。执行时，实际等级 = 基础等级 + 当前战术优势层数 × 0.5。速度受神机加成：实际速度 = 基础速度 + 神机层数 × 0.5。
>
> 策略执行后对缓冲的消耗规则：消耗的连续指令构成该策略的序列长度。例如 TT 消耗缓冲中连续 2 个 T；TTA 消耗连续 3 个指令（T、T、A）。剩余指令保持原有顺序。

### 长度 1：基础指令

始终可用，作为保底。

| 序列 | 名称 | 效果 | 速度 |
|------|------|------|------|
| A | 炮击 | 1 级钝击，1 名其他存活玩家 | 1 |
| D | 布防 | 1.5 级格挡 | 1 |
| T | 谋划 | 指令存入缓冲 | 1 |

### 序列学习法则

玩家无需背诵全部序列表。理解以下三条法则即可推断绝大多数序列的效果：

**法则一：重复强化**——同一指令连续重复，每多一次效果显著增强。A 序列提升攻击等级，D 序列提升格挡等级，T 序列每多一个产出更持久的战术优势并在特定长度解锁永久 Buff。

**法则二：两两配合**——任意两种不同指令的组合都有独一无二的身份。攻击+防守 = 稳步推进；防守+攻击 = 盾反；攻击+战术 = 校准标记并获得战术优势；战术+攻击 = 谋定后动（更强更快）；防守+战术 = 防御中积蓄并获得战术优势；战术+防守 = 优势布防（更强更快）。

**法则三：三段定式**——三条指令的序列中，T 的位置决定策略性质。T 在首位 = 战术准备后放大后续行动；T 在末位 = 行动后获得战术优势；T 在中间 = 行动中穿插战术调整产生控制效果。纯粹的三连 A / 三连 D / 三连 T 则是法则一的延伸。

| 场景 | 处理 |

## 设计定位

拿破仑是全游戏最特殊的角色——他完全脱离气/蓄力经济，以时间为唯一资源。核心循环为：发布指令填充缓冲 → 择机消耗缓冲中的连续子序列执行策略 → 策略产出有时限的战术优势 → 在窗口期内用强化后的指令和策略压制对手 → 战术优势到期前重新执行策略刷新。

**三阶段节奏**：

1. **开局（积攒期）**：用 T 和 D 填充缓冲。防御指令保障生存，战术指令为策略序列提供必要组成。此时没有任何被动增益——必须主动执行策略才能获得战术优势。
2. **中期（爆发期）**：缓冲填至 3–4 条后，择机执行 TT 或 TTT 获取战术优势窗口，在其持续时间内用 A/AA/AAA 等攻击策略打出强化伤害。TTA 是"收割"型选择——消耗全部战术优势换取决定性一击，代价是之后几回合需要重新积累。
3. **后期（传奇期）**：若能完成 TTTTT 获得皇帝引擎，拿破仑进入 5 回合高强度期（每回合 +2 战术优势）。此时可择机释放 TTAA——以目标为中心的范围毁灭，消耗全部战术优势同时清场。TATAT 解锁变身提供退路和永久低功率引擎，确保拿破仑在皇帝到期后仍能持续运营。

**优势**：零资源消耗，战术灵活；战术优势窗口期内攻防两端同时强化；神机提供永久速度优势让拿破仑在关键时刻抢先出手。

**风险**：必须在"保留缓冲指令以维持策略可执行性"和"消耗指令执行策略以获取增益"之间持续权衡。战术优势到期后若未能重新执行策略刷新，拿破仑回到裸体状态（A 仅 1 级、D 仅 1.5 级），极易被压制。TTA 和 TTAA 消耗全部战术优势的代价真实存在——消耗后需要重新执行策略积累。

**反制窗口**：拿破仑在战术优势到期且缓冲不足以执行新策略时最为脆弱——此时攻击和防御均为基础值。对手应在拿破仑执行 TT/TTT 之前（即缓冲仅有 1–2 条指令时）集中施压，阻止其安全获取战术优势窗口。此外，TTA 和 TTAA 消耗全部战术优势后是一个明显的疲软期，可以利用。
