# 叶倾仙

> 远古仙人的第二世，以生命为代价换取资源，在消耗与收割之间建立循环。
>
> 通用规则（等级结算、生命结构、格挡等）见 [基础规则手册](../基础规则手册.md)。资源与 Buff 速查见 [附录](../附录/)。

## 属性

- **形态名称**：仙古形态
- **生命结构**：2 HP（健康 / 濒死 / 死亡）

## 被动技能

### 祭道

- **祭道计数**：角色级计数器，记录叶倾仙本局累计获得的气与蓄力总量。每次获得气或蓄力（含使用气、蓄力、吞天、进入濒死、凹类技能等任意来源）时，祭道计数增加对应数量。祭道计数无上限。
- **激活条件**：祭道计数 ≥ 3 时生效。激活后为永久生效，即使之后祭道计数继续增长也不影响激活状态。
- **效果**：使用技能支付消耗时，若某项资源的缺口为 1 单位，可将自身健康状态左移 1 层以抵免该缺口。每次使用技能至多触发 1 次。
- **抵免规则**：祭道的左移直接填补资源缺口，不经过「获得资源 → 再支付」两步。支付完成后资源余额不增加。
- **与受伤的关系**：祭道造成的左移独立于其他玩家造成的伤害。因祭道从健康进入濒死时，正常获得进入濒死的 1 气；恢复健康后再次以祭道进入濒死，可以再次获得。该气在本次技能支付完成后结算，不会被同一次支付消耗。

### 吞天

- **左移回馈**：每当叶倾仙使用的任意技能（含基础技能）直接使 1 名其他存活玩家健康状态左移 1 层时，叶倾仙获得 1 气或 1 蓄力（由叶倾仙在提交时选择）。只有实际发生左移时才会触发。君临天下不造成伤害，不触发此效果。
- **击杀学习**：当叶倾仙使用的技能直接使 1 名其他玩家进入死亡状态时，在该回合结算完成后，叶倾仙可以从该玩家的当前角色中学习 1 个专属技能（攻击 / 防御 / 资源 / 特殊分类）或 1 个被动技能，也可以放弃本次学习。已学技能为角色级，角色切换后保留、切回时恢复。已学技能无数量上限，从下回合起可用。不可学习变身技能和基础解锁技能（拳 / 斩 / 治疗）。若所学的被动技能引用叶倾仙不拥有的资源或机制，则其中无法生效的部分无效果。

## 变身收益

保留全部基础招式（气、蓄力、凹、紫翼双凹、剁、挡、超防、变身），额外解锁：

拳、斩、治疗、**掌仙术**、**君临天下**。

## 专属技能

### 掌仙术

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 3 气 |
| 速度 | 1 |
| 等级 | 3 |
| 伤害类型 | 魔法 |
| 目标 | 1 名其他存活玩家（预选） |

**结算说明**：

1. 对目标造成 3 级魔法伤害，按通用攻击结算流程进行技能对抗与伤害结算。

### 君临天下

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 1 气 + 2 蓄力 |
| 速度 | 2 |
| 等级 | 3 |
| 目标 | 1 个地块（预选），并对该地块及顺、逆时针各 2 格内的所有其他存活玩家（自动） |

**结算说明**：

1. 君临天下为攻击分类，但不造成伤害。君临天下的有效伤害等级恒为 0。
2. 君临天下参与技能对抗：若对抗中技能差 ≥ 0.5，则君临天下命中；若技能差 < 0.5，则君临天下不命中。
3. 君临天下命中时，对范围内所选招式等级 ≤ 3 的其他存活玩家施加「恐惧」Buff：玩家级，持续 2 回合；持有者只能使用「气」分类技能；受到攻击时，攻击方技能等级与伤害等级均 +1。
4. 若同一目标在君临天下范围和自身 Dominion 范围中均被覆盖，恐惧的施加仅结算 1 次，并使用 Dominion 的更高等级判定（参照 AGENTS.md 中 Collapsing Fear 规则）。
5. 君临天下不命中时，不对任何目标施加恐惧。
6. 命中后不造成伤害，不令目标健康状态左移，不触发吞天的左移回馈。
7. 重复施加恐惧时，持续时间取当前剩余时间与新施加之 2 回合中的较大值。

## 设计定位

叶倾仙围绕祭道的自伤换资源和吞天的左移回馈建立循环。前几回合通过气、蓄力积累祭道计数至 3 激活祭道，之后可以超支使用高消耗技能——每次超支以 1 层健康状态左移为代价。掌仙术提供 3 级单体魔法攻击作为主要输出手段；君临天下为无伤害的范围攻击，在对抗获胜时对范围内低等级目标施加恐惧进行控场。包括拳、斩、剁等基础技能在内的任意招式只要实际令其他玩家左移，都可通过吞天回收资源。击杀学习让叶倾仙在收割对手后获得被击杀者的技能，随着对局推进逐步扩展技能树，形成滚雪球效应。

**核心风险**：祭道自伤直接消耗生命，在濒死状态下使用祭道等于自杀。叶倾仙必须在资源积累和生命保持之间权衡，不能被凹打断资源节奏时陷入被动。

**反制窗口**：
- 在叶倾仙积累祭道计数至 3 之前，其资源结构不稳定，凹和紫翼双凹可以干扰其资源累积；
- 君临天下需要对抗获胜才能施加恐惧，使用高等级技能（≥ 3.5）的目标可以反制其控场；
- 掌仙术消耗 3 气，高消耗意味着低频率——若叶倾仙连续使用掌仙术，必然频繁触发祭道自伤，为对手提供了反杀窗口；
- 恐惧状态下虽受限但仍可使用气获取资源，并非完全无解。
