# 叶倾仙

> 远古仙人的第二世，以生命为代价换取资源，在消耗与收割之间建立循环。
>
> 通用规则（等级结算、生命结构、格挡等）见 [基础规则手册](../基础规则手册.md)。资源与 Buff 速查见 [附录](../附录/)。

## 属性

- **形态名称**：仙古形态
- **生命结构**：2 HP（健康 / 濒死 / 死亡）

## 被动技能

### 祭道

- **激活条件**：当前持有气与蓄力的合计值 ≥ 3 时生效。未达到时不触发。
- **效果**：使用技能支付消耗时，若某项资源缺口为 1 单位，可将自身健康状态左移 1 层以抵免该缺口。每次使用技能至多触发 1 次。
- **与受伤的关系**：祭道的自伤左移独立于其他玩家造成的伤害。同一次健康状态左移不能同时用于祭道和受伤触发（如进入濒死获得 1 气）。

### 吞天

- 每当叶倾仙使用的任意招式（包括基础招式）使 1 名其他存活玩家健康状态左移 1 层，叶倾仙获得 1 气或 1 蓄力（由叶倾仙在提交时选择）。只有实际发生左移时才会获得资源。
- 击杀其他玩家时，可学习该玩家的任一技能（暂不实现，后续开发跟进）。

## 变身收益

保留全部基础招式（气、蓄力、凹、紫翼双凹、剁、挡、超防、变身），额外解锁：

拳、斩、治疗、**掌仙术**、**君临天下**。

## 专属技能

### 掌仙术

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 3 气 |
| 速度 | 1 |
| 等级 | 3 |
| 伤害类型 | 魔法 |
| 目标 | 1 名其他存活玩家（预选） |

### 君临天下

| 属性 | 值 |
|------|-----|
| 分类 | 特殊 |
| 消耗 | 1 气 + 2 蓄力 |
| 速度 | 2 |
| 等级 | 3 |
| 目标 | 1 个地块（预选），并对该地块及顺、逆时针各 2 格内的所有其他存活玩家（自动） |

**结算说明**：

1. 君临天下不造成伤害，也不会直接令目标健康状态左移。
2. 对范围内本回合所选招式等级 ≤ 3 的其他存活玩家施加「恐惧」Buff（玩家级，持续 2 回合）：该玩家只能使用气；受到攻击时攻击方等级 +1。

## 设计定位

叶倾仙围绕祭道的自伤换资源和吞天的左移回馈建立循环。前期积累资源至合计 ≥ 3 激活祭道，之后可以超支使用高消耗技能。掌仙术提供 3 级单体攻击，君临天下则选择地块施放无伤害的范围恐惧控制。包括基础攻击在内的任意招式只要实际令其他玩家左移，都可通过吞天回收资源。主要风险在于祭道自伤会加速自身死亡——在濒死状态下使用祭道等于自杀。反制窗口在于叶倾仙积累资源的回合可用凹打断。
