# 魑魅

> 当前仅可在训练房使用，标准房暂时禁用。

> 幽冥行者，以魂为资、操控对手，通过削弱与度化掌控战局。
>
> 通用规则（等级结算、生命结构、格挡等）见 [基础规则手册](../基础规则手册.md)。资源与 Buff 速查见 [附录](../附录/)。

## 属性

- **形态名称**：幽冥形态
- **生命结构**：2 HP（健康 / 濒死 / 死亡）

## 被动技能

### 地狱行者

- **触发条件**：魑魅使用不造成伤害的非基础招式时触发。
- **效果**：除触发该效果的魑魅本人外，该招式所选目标以及棋盘上离该目标最近的 2 名存活玩家（按双方地块索引最短路径计算；距离相同时顺时针优先）在下两回合内使用攻击分类招式时，总消耗 +1（由该玩家自选多支付哪种资源）。其他魑魅仍可受到该效果；同一玩家在一次触发中只会被选择一次；资源不足时该攻击招式不可提交。
- **不叠加**：同一技能产生的此效果不叠加——若目标已持有地狱行者效果，再次触发仅刷新持续时间至 2 回合。
- **持续时间**：2 回合，从下回合开始生效。

### 幽怨

- **标记分配**：每回合选择阶段开始时，检测除产生该标记的魑魅本人外所有存活玩家当前持有的全部资源总量（气 + 蓄力 + 辉星等）。其他魑魅仍可被标记。总量最高的玩家获得「幽怨标记」Debuff（玩家级）。全场同时至多存在 1 个幽怨标记。若并列最高，优先比较气持有量；仍并列则选择顺时针方向第一个满足要求的玩家。
- **攻击加成**：魑魅对持有幽怨标记的玩家使用攻击分类招式时，攻击等级 +0.5。
- **获取加成**：魑魅从持有幽怨标记的玩家处获得资源时，单次获得的资源量 +1。若为气或蓄力，对方额外损失 1 点同种资源（余额不足则不触发额外损失）；若为魂，额外获得 1 魂（凭空生成）。

## 核心机制：魂

魂是魑魅专属资源（角色可见，切走后保留、切回恢复）：

- **获得**：夺魂获得 1 魂；摄魄获得 1 魂，本回合至少被一个攻击行动选择时额外获得 1 魂。
- **用途**：魂层数决定恫吓的伤害等级（n）和度神决的速度（n）；度化期间可消耗魂为被度化玩家补充资源（1 魂补充 1 点任意资源）。
- **上限**：无上限。

## 变身收益

保留全部基础招式，额外解锁：

拳、斩、治疗、**夺魂**、**摄魄**、**恫吓**、**度神决**。

## 专属技能

### 夺魂

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 1 气 |
| 速度 | 2 |
| 等级 | 2 |
| 伤害类型 | —（不造成伤害） |
| 目标 | 1 名其他存活玩家（预选） |

**结算说明**：

1. 夺魂参与技能对抗。若目标使用非防御技能且对魑魅有效，进行标准技能对抗（夺魂等级 2）。
2. 无论对抗结果如何，夺魂不造成伤害。
3. 魑魅获得 1 层「魂」。
4. 对目标施加「夺魂」Debuff（玩家级，持续 2 回合，从下回合开始生效）：该玩家伤害等级 −0.5，速度 −1。速度 −1 可与其他减速效果叠加。
5. 同一技能的效果不可叠加——若目标已持有夺魂 Debuff，再次施加仅刷新持续时间至 2 回合。
6. 夺魂不造成伤害，触发地狱行者。

### 摄魄

| 属性 | 值 |
|------|-----|
| 分类 | 防御 |
| 消耗 | 1 蓄力 |
| 速度 | 4 |
| 等级 | +∞ |
| 目标 | 自身（自动） |

**结算说明**：

1. 本回合免疫任何攻击。
2. 魑魅获得 1 层「魂」。
3. 若本回合有至少 1 名其他存活玩家以魑魅为目标使用了攻击分类招式（无论攻击是否最终命中），魑魅额外获得 1 层「魂」。每回合至多额外获得 1 层。
4. 摄魄不造成伤害，触发地狱行者。

### 恫吓

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 1 气 + 1 蓄力 |
| 速度 | 2 |
| 等级 | n |
| 伤害类型 | 魔法 |
| 目标 | 1 名其他存活玩家（预选） |

其中 n = 魑魅当前持有的魂。

**结算说明**：

1. 若目标本回合使用了资源分类招式，伤害等级 = n + m，其中 m = 目标本回合提交时持有的全部资源总数（气 + 蓄力 + 辉星等特殊资源）。
2. 若目标本回合未使用资源分类招式，伤害等级 = n。
3. 恫吓的伤害不归属为魑魅造成的伤害——不触发吞天等「造成伤害时」的被动效果。目标仍可正常使用防御技能和进行技能对抗。
4. 恫吓造成伤害，不触发地狱行者。

### 度神决

| 属性 | 值 |
|------|-----|
| 分类 | 特殊 |
| 消耗 | X（任意资源，后发选择 X 值与目标） |
| 速度 | n |
| 等级 | X − 1 |
| 目标 | 1 名其他存活玩家（后发） |

其中 n = 魑魅提交度神决时持有的魂（按速度 0～4 截断），X 为魑魅在后发阶段选择的任意正整数。X 不得超过后发支付时持有的资源总数（气 + 蓄力 + 魂）。

**结算说明**：

1. 后发阶段选择 X 值与目标。消耗 X 点任意资源（从气、蓄力、魂中支付）。
2. 进行条件判定：比较目标本回合所选招式对魑魅的有效技能等级（若目标招式对魑魅无效，则视为 0）与 X：
   - 若目标等级 < X：目标进入「度化」状态（玩家级，从下回合开始生效）。
   - 若目标等级 ≥ X：度化失败，无任何效果。已消耗的 X 点资源不退还。
3. **度化期间**：魑魅为被度化玩家选择每回合的行动。被度化玩家仍正常消耗自身资源、正常受到攻击、正常参与结算。
4. **累计消耗**：度化期间被度化玩家每次行动的消耗计入累计值。累计值达到 X 后，度化立即结束。

   累计消耗计算规则：

   | 被操控技能类型 | 累计消耗 | 附加条件 |
   |--------------|---------|---------|
   | 自身产生脆弱效果的技能（如拳（默认）、紫翼双凹、治疗） | 4 | X ≥ 3 |
   | 自身特性取消脆弱效果的技能（如泥岩的拳） | 2 | — |
   | 无消耗且无伤害的技能（如凹） | 1 | — |
   | 其他技能 | 技能的实际资源消耗 | — |

   「实际资源消耗」为该技能消耗的气与蓄力总和；零消耗技能若不在上述特殊列表中（如蓄力、变身、气），按对应规则计算。

5. **魂补充**：度化期间，魑魅可消耗魂为被度化玩家补充资源（1 魂兑换 1 点任意资源，在选择被度化玩家行动时支付）。此操作不计入累计消耗。
6. 度神决不造成伤害，触发地狱行者。

## 设计定位

魑魅是控制型角色，核心机制分为两个层次：

**魂经济**：夺魂和摄魄是获取魂的主要手段。夺魂同时削弱目标的伤害输出和速度，摄魄提供无敌防御并获取魂。二者均触发地狱行者，叠加对手的攻击消耗压力。

**终结技**：恫吓和度神决消耗魂兑现收益。恫吓的伤害随魂层数成长，对资源型角色（如储君、叶倾仙）有额外打击。度神决是高风险高回报的终极控制——成功后完全操控对手，但失败则血本无归。

主要弱点：
- 魂的获取依赖夺魂（需参与技能对抗）和摄魄（需对手攻击魑魅），对手可以刻意不攻击魑魅来限制魂的增长；
- 地狱行者和幽怨标记需要时间积累压力，快节奏对局中难以发挥；
- 度神决若 X 选择过高则消耗巨大、选择过低则容易被对手技能等级压制而失败；
- 魑魅自身没有直接的高伤害手段（恫吓依赖魂和对手资源状态），终结能力偏弱。
