# 星神

> 星辰之力的化身，以神体减免伤害、和光同尘叠加护盾，最终以创生星核进入超脱状态掌控对局。
>
> 通用规则（等级结算、生命结构、格挡等）见 [基础规则手册](../基础规则手册.md)。资源与 Buff 速查见 [附录](../附录/)。

> **测试限制**：星神当前只在练功房角色配置中开放。普通房间不会显示星神变身入口，服务端也会拒绝普通房间变身为星神。

## 属性

- **形态名称**：星神形态
- **生命结构**：2 HP（健康 / 濒死 / 死亡）

## 被动技能

无。

## 核心机制：神体

神体是星神专属的永久伤害减免层，以 Buff 层数表示（角色级，切走保留、切回恢复）：

- **获得**：每次使用和光同尘获得 0.5 神体；超脱状态下每回合获得 0.5 神体。支持 0.5 增量，无上限。
- **效果**：星神受到非真实伤害时，先将伤害等级减去当前神体等级，再按剩余等级进行防御结算。神体减免先于格挡和防御等级计算，不影响技能对抗。
- **真实伤害**：裂空刺等真实伤害无视神体减免。

## 变身收益

保留全部基础招式，额外解锁：

拳、斩、治疗、**和光同尘**、**星云震荡**、**创生星核**。

## 专属技能

### 和光同尘

| 属性 | 值 |
|------|-----|
| 分类 | 防御 |
| 消耗 | 1 气 |
| 速度 | 2 |
| 等级 | n（格挡） |
| 目标 | 自身（自动） |

其中 n = 本局累计使用和光同尘的次数，无上限。

**结算说明**：

1. 生成 n 级临时格挡（重新生成型——仅当次施放有效，破碎不延续到下次施放）。
2. 获得 0.5 神体（永久积累）。
3. 若处于超脱状态，此次获得的神体翻倍（即 1.0 神体）。

### 星云震荡

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 1 蓄力 |
| 速度 | 1 |
| 等级 | 1 |
| 伤害类型 | 魔法 |
| 目标 | 1 名其他存活玩家（预选） |

**结算说明**：

1. 对所选玩家造成 1 级魔法伤害。
2. 对该玩家施加「震荡」Debuff（玩家级，持续 2 回合）：持有震荡的玩家若在接下来的 2 回合内使用攻击分类招式，则在那个回合结束时自身健康状态左移 1 层。

### 创生星核

| 属性 | 值 |
|------|-----|
| 分类 | 特殊 |
| 消耗 | 3 气 + 2 蓄力 |
| 速度 | 2 |
| 等级 | 0 |
| 目标 | 自身（自动） |

**结算说明**：

1. **当回合格挡**：本回合获得 3 级临时格挡。
2. **进入超脱**：获得「超脱」Buff（角色级，持续 4 回合）。超脱状态下每回合自动获得：
   - 攻击等级 +0.5；
   - 神体 +0.5；
   - 速度 +1。
   - 和光同尘叠加的神体翻倍。
3. **复活甲**：处于超脱状态时，若受到致命伤害，改为退出超脱状态并将自身健康状态设为濒死（即免疫此次死亡）。
4. **主动选择**：超脱持续的 4 回合内，任意回合可选择以下一项执行：
   - **融合**：将超脱变为永久（无回合限制，其余效果不变）。永久超脱下复活甲每次触发后退出超脱并存活，之后可再次使用创生星核重新进入。
   - **引爆**：结束超脱，对所有其他存活玩家造成魔法伤害。引爆等级 = 3 + 超脱期间已累计的攻击等级加成，引爆速度 = 2 + 超脱期间已累计的速度加成。
5. 若 4 回合内未主动选择融合或引爆，超脱自然结束（无额外效果）。

## 设计定位

星神是防御型成长角色。核心循环：使用和光同尘叠加神体和护盾 → 积累 3 气 2 蓄力 → 创生星核进入超脱 → 选择融合或引爆终结对局。神体提供先于防御结算的被动伤害减免，使星神在护盾和格挡之外拥有第三层防护。主要风险在于前期资源积累缓慢，和光同尘初期护盾等级低、易被击穿，且超脱的四回合等待期是对手集中火力的窗口。星云震荡提供低成本骚扰，可威慑对手不敢随意使用攻击技能。
