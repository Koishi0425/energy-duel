# 星神

> 星辰之力的化身，以神体减免伤害、和光同尘叠加护盾，最终以超脱状态掌控对局。
>
> 通用规则（等级结算、生命结构、格挡等）见 [基础规则手册](../基础规则手册.md)。资源与 Buff 速查见 [附录](../附录/)。

> 星神现已在普通房间和练功房开放，可通过标准变身入口选择。

## 属性

- **形态名称**：星神形态
- **生命结构**：2 HP（健康 / 濒死 / 死亡）

## 被动技能

无。

## 核心机制：神体

神体是星神专属的永久伤害减免层，以 Buff 层数表示（角色级，切走保留、切回恢复）。神体属于防御类型的第一层减免，详见《基础规则手册》第 4.2 节。

- **获得**：每次使用和光同尘获得 0.5 神体；超脱状态下每回合获得 0.5 神体。支持 0.5 增量，无上限。
- **效果**：结算时在格挡之前先将伤害等级减去当前神体等级。神体减免不影响技能对抗。
- **交互**：穿刺伤害不忽略神体；真实伤害忽略神体。

## 变身收益

保留全部基础招式，额外解锁：

拳、斩、治疗、**和光同尘**、**震荡**、**超脱**、**空心拳**。

## 专属技能

### 和光同尘

| 属性 | 值 |
|------|-----|
| 分类 | 防御 |
| 消耗 | 1 气 |
| 速度 | 2 |
| 等级 | n（格挡） |
| 目标 | 自身（自动） |

其中 n = 本局累计使用和光同尘的次数，无上限。

**结算说明**：

1. 生成 n 级临时格挡（重新生成型——仅当次施放有效，破碎不延续到下次施放）。
2. 获得 0.5 神体（永久积累）。
3. 若处于超脱状态，此次获得的神体翻倍（即 1.0 神体）。

### 震荡

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 1 蓄力 |
| 速度 | 1 |
| 等级 | 1 |
| 伤害类型 | 魔法 |
| 目标 | 1 名其他存活玩家（预选） |

**结算说明**：

1. 对所选玩家造成 1 级魔法伤害。
2. 对该玩家施加「震荡」Debuff（玩家级，持续 2 回合）：持有震荡的玩家若在接下来的 2 回合内使用攻击分类招式，则在那个回合结束时自身健康状态左移 1 层。

### 超脱

| 属性 | 值 |
|------|-----|
| 分类 | 特殊 |
| 消耗 | 2 气 + 2 蓄力 |
| 速度 | 2 |
| 等级 | 3 |
| 目标 | 自身（自动） |

**结算说明**：

1. **当回合格挡**：本回合获得 3 级临时格挡。
2. **进入超脱**：获得「超脱」Buff（角色级，持续 4 回合）。超脱状态下，每回合结束时依次获得：
   - 健康状态右移 1 层；
   - 攻击等级 +0.5（累积）；
   - 神体 +0.5（累积）；
   - 速度 +1（累积）；
   - 和光同尘叠加的神体翻倍。
3. **被动保护**：处于超脱状态时，若在主动选择融合或引爆之前受到致命伤害，改为将自身健康状态左移 1 层，然后退出超脱状态。在健康状态下触发时进入濒死并存活；在濒死状态下触发时直接死亡。
4. **主动选择**：超脱持续的 4 回合内，任意回合可选择以下一项执行（只能选择一次）：
   - **融合**：将超脱变为永久（「永久超脱」Buff，角色级，无回合限制）。永久超脱下，每回合结束时仍获得攻击等级 +0.5、神体 +0.5、速度 +1 与健康状态右移 1 层。受到致命伤害时，改为退出永久超脱并将自身健康状态设为濒死（即免疫此次死亡）。退出永久超脱后，可再次使用超脱重新进入。
   - **引爆**：结束超脱，对所有其他存活玩家造成魔法伤害。引爆等级 = 3 + 超脱期间已累计的攻击等级加成，引爆速度 = 2 + 超脱期间已累计的速度加成。
5. 若 4 回合内未主动选择融合或引爆，超脱自然结束（无额外效果）。

### 空心拳

| 属性 | 值 |
|------|-----|
| 分类 | 攻击 |
| 消耗 | 2 气 + 2 蓄力 |
| 速度 | 2 |
| 等级 | 3 |
| 伤害类型 | 穿刺 |
| 目标 | 1 名其他存活玩家（预选） |

**结算说明**：

1. 穿刺伤害在结算时无视目标的格挡等级与护甲等级——跳过格挡与护甲的减免步骤。
2. 穿刺伤害不无视神体减免与屏障。
3. 除此之外遵循标准技能对抗与伤害结算规则。

## 设计定位

星神是防御型成长角色。核心循环：使用和光同尘叠加神体和护盾 → 积累 2 气 2 蓄力 → 超脱进入成长状态 → 选择融合永久化或引爆终结对局。超脱新增的每回合自动回血大幅增强了生存能力，但被动保护机制（左移一层退出）在濒死状态下无法免死，要求星神在健康状态下谨慎规划主动选择的时机。空心拳提供无视格挡和护甲的穿刺伤害，作为对抗高防御角色的突破手段，但仍会受到神体与屏障影响。主要风险在于前期资源积累缓慢，和光同尘初期护盾等级低、易被击穿，且超脱的被动保护不如原版的完全免死可靠。震荡提供低成本骚扰，可威慑对手不敢随意使用攻击技能。
