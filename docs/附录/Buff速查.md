# Buff 速查

> 版本：v15（依据 `shared/config/game.json` 整理）
>
> 归属为「角色」的 Buff 切走角色时后台保留、持续时间继续扣除、切回时恢复。归属为「玩家」的 Buff 跨角色持续生效。

## 通用 Buff

| Buff ID | 名称 | 归属 | 持续时间 | 说明 |
|---------|------|------|----------|------|
| fragile | 脆弱 | 角色 | — | 受到任何有效伤害或状态左移直接死亡 |
| defend_broken | 挡已破碎 | 玩家 | 本局 | 挡的防御等级降为 0 |
| swayed | 动摇 | 玩家 | 1 回合 | 下回合速度 -1 |
| calibrated | 校准标记 | 玩家 | 1 回合 | 下回合受到攻击时攻击方等级 +1 |
| attack_forbidden | 全面压制 | 玩家 | 1 回合 | 不能提交攻击分类技能 |
| defense_forbidden | 崩裂禁防 | 玩家 | 2 回合 | 不能提交防御分类技能 |
